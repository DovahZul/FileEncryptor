RichTextBox.LoadFile - метод (String, RichTextBoxStreamType)

.NET Framework (current version) Другие версии 
 
Загружает в элемент управления RichTextBox определенный тип файла.
Пространство имен:   System.Windows.Forms
Сборка:  System.Windows.Forms (в System.Windows.Forms.dll)

Синтаксис
C#C++F#VB
public void LoadFile(
	string path,
	RichTextBoxStreamType fileType
)
Параметры
path
Имя и расположение файла, который будет загружен в элемент управления.
fileType
Одно из значений RichTextBoxStreamType.
Исключения
Exception	Condition
IOException	
При загрузке файла в элемент управления произошла ошибка.
ArgumentException	
Загружаемый файл является документом RTF.
Заметки
При загрузке файла с помощью метода LoadFile содержимое загружаемого файла замещает все содержимое элемента управления RichTextBox.Это приводит к тому, что значения свойств Text и Rtf меняются.С помощью этого метода можно загрузить предварительно созданный текст или RTF-документ в элемент управления для дальнейшей с ним работы.Если нужно сохранить файл, можно воспользоваться методом SaveFile.
Эта версия метода LoadFile позволяет указать тип загружаемого файла.Эта возможность позволяет загружать в элемент управления иные, чем RTF-документы, файлы.
System_CAPS_noteПримечание
Метод LoadFile не откроет файл, пока не будет создан дескриптор для элемента управления RichTextBox.Перед тем как вызвать метод LoadFile, убедитесь, что создан дескриптор элемента управления.
